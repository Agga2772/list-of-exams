Exam List Application  

This is a simple web application that displays a list of exam sessions, allowing users to filter the list based on exam title, location, and candidate name.  

Features  
- Backend: A Node.js/Express server that serves exam session data via an API.  
- Frontend: A simple HTML, CSS, and JavaScript interface that fetches and displays exam data.  
- Filtering: Users can filter exams by title, location, and candidate name.  
- Sorting: Exam sessions are displayed in date order.  

Technologies Used  
- Backend: Node.js, Express.js, CORS  
- Frontend: HTML, JavaScript, CSS
